# 🎮 Gerenciador de Peças - Tetris em C

Projeto desenvolvido como **atividade da disciplina de Laboratório de Programação em C**, com objetivo de praticar **lógica de programação**, **listas encadeadas**, **estruturas condicionais** e **funções**.

---

## 📚 Descrição do Projeto

O programa simula um **gerenciador de peças de Tetris**, permitindo:

- Cadastrar peças com tipo, cor e borda;  
- Listar todas as peças cadastradas;  
- Reorganizar a lista de peças seguindo uma ordem pré-definida: T, S1, S2, I, L1, L2, Q.

O projeto utiliza **listas encadeadas dinâmicas** para armazenar as peças.

---

## 🧠 Conceitos Trabalhados

- Lógica de programação  
- Estruturas de dados: listas encadeadas  
- Linguagem C  
- Funções  
- Estruturas condicionais (`if`, `switch`)  
- Loops (`while`)  
- Validação de entrada do usuário  
- Alocação e liberação de memória (`malloc` e `free`)  

---

## ⚙️ Como Executar

1. Compile o programa:

gcc pecas.c -o pecas

2. Execute:

./pecas

3. Siga o menu interativo para cadastrar, listar ou reorganizar peças.
